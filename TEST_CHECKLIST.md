### 9. API 测试
使用 curl 或 Postman 测试：

#### GET 请求
```bash
curl http://localhost:3000/api/menu
```
- [ ] 返回 JSON 数据
- [ ] 包含 `items` 和 `lastModified` 字段

#### POST 请求
```bash
curl -X POST http://localhost:3000/api/menu \
  -H "Content-Type: application/json" \
  -d '{"items":[{"id":"test","category":"Test","name":"API Test","price":5}]}'
```
- [ ] 返回成功响应
- [ ] `data/menu.json` 被更新

#### DELETE 请求（单个）
```bash
curl -X DELETE "http://localhost:3000/api/menu?id=test"
```
- [ ] 返回成功响应
- [ ] 菜品被删除

#### DELETE 请求（全部）
```bash
curl -X DELETE http://localhost:3000/api/menu
```
- [ ] 返回成功响应
- [ ] `data/menu.json` 被清空
